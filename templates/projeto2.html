Verificador de Senhas

Diversas empresas como Facebook, Yahoo, entre outras, já foram invadidas e sua base de dados com usuários e senhas vieram ao conhecimento público. O site “Have I been Pwned?”, traduzindo para português "Fui sacaneado?" permite consultar se uma senha já foi divulgada através da consulta em uma base de dados com 555 milhões de senhas violadas anteriormente.
Mas qual a importância disso? Os hackers simplesmente utilizam listas com milhões de e-mails e senhas já conhecidos após violações e executam o chamado ataque de força bruta, no qual, basicamente percorrem todos os e-mails e senhas presente neste banco de dados visando realizar login em um determinado site. Desta forma, caso seu e-mail e senha já tenha sido alvo de violações, você pode estar vulnerável e estar sujeito a um futuro problema caso consigam fazer login em alguma de suas contas em um eventual ataque. O presente script utiliza uma função em Python que codifica a senha que será consultada utilizando a criptografia MD5 e realiza a busca apenas dos 5 primeiros caracteres da criptografia. Visto que, embora o site seja seguro, como estamos realizando uma consulta na internet, durante este percurso a senha pode ser interceptada, utilizando este método de criptografia a busca se torna ainda mais segura. Caso tenha interesse em saber um pouco mais sobre a criptografia MD5, a DevMedia explica sobre o assunto aqui.

Como utilizar o script?

Para utilizar o script é necessário abrir o Prompt de Comando. Após abri-lo acesse a pasta na qual está o arquivo python, utilize o seguinte comando:

cd “C:\Users\gabriel\Desktop\teste”

Neste caso, o arquivo python está localizado em minha área de trabalho, em uma pasta denominada teste.
Agora execute o arquivo python e digite as senhas a serem verificadas:

python verificador_de_senha.py senha123 tartaruga9090 br4s1l3450

Foi executado o arquivo python "verificador_de_senha.py" e verificado as respectivas senhas: "senha123", "tartaruga9090" e "br4s1l3450".
