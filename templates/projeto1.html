Relações hipsométricas

A relação entre o diâmetro e altura das árvores é denominada “relação hipsométrica”. Nos inventários florestais a medição da altura de todas as árvores elevaria o custo de trabalho, uma vez que demandaria muito tempo, além de possíveis erros ocasionados por dificuldade de visibilidade de copa, erros do operador, entre outros. Desta forma, comumente mede-se o diâmetro à 1,3m do solo de todas as árvores da parcela e mede-se a altura de apenas algumas árvores, as demais são estimadas através de equações hipsométricas.
O ajuste de uma equação hipsométrica pode ser realizado facilmente no Excel, R e Python. Porém, como é necessário realizar o ajuste de diversas equações visando compará-las segundo critérios estatísticos para seleção da melhor equação para seu conjunto de dados, esse processo pode demandar alguns minutos e em alguns casos horas.
Logo, o presente script em Python, realiza de maneira automatizada em apenas segundos as seguintes tarefas:

• Teste de normalidade da variável altura;
• histograma de frequência da altura;
• ajuste das equações presentes na tabela abaixo;

• predição das alturas utilizando as equações ajustadas anteriormente;
• gráficos de tendência e resíduos das equações;
• cálculo do coeficiente de determinação ajustado (R² ajustado), erro padrão da estimativa (syx) e erro padrão da estimativa em porcentagem (syx %);
• salva todo o processamento em uma planilha e salva os gráficos como imagem.

Como utilizar o script?

O arquivo com a base de dados utilizada para o ajuste das equações deve estar em formato em csv, delimitado por ponto e vírgula (;) nomeado como “dados”. A coluna com a numeração, diâmetro a 1,3m e altura das árvores devem ser nomeadas: “arvore”, “dap” e “altura”, respectivamente. O arquivo com a base para o ajuste e arquivo em Python devem estar na mesma pasta. Confira o exemplo nas imagens abaixo:

Arquivo "dados.csv" e "ajuste.py"

Nomeação das colunas

Para execução do script é necessário instalar as seguintes bibliotecas: Numpy, Pandas, Matplotlib, Scipy, Statsmodels e XlsxWriter. Ressaltando que é necessário realizar a instalação das bibliotecas apenas uma vez, na primeira utilização do script. Abra o Prompt de Comando e execute os comandos a seguir:

python -m pip install numpy

pip install pandas

python -m pip install -U matplotlib

pip install scipy

pip install -U statsmodels

pip install XlsxWriter

Após instalar todas as bibliotecas, acesse a pasta na qual está o arquivo python e a base dados que será utilizada para ajuste das equações, utilize o seguinte comando:

cd “C:\Users\gabriel\Desktop\teste”

Neste caso, o arquivo python está localizado em minha área de trabalho, em uma pasta denominada teste.

Agora execute o arquivo python:

python ajuste.py
